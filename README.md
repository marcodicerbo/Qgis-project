# Qgis-project
Landslide inventory map of the municipality of Deliceto in Puglia, Italy

![carta frane deliceto](https://github.com/user-attachments/assets/7f0f57f9-fc3d-40d0-bff8-aac43971585b)

This map shows the location, extent and types of landslides that occurred in the municipality of Deliceto in Puglia, Italy.
Coordinate reference system: EPSG:32633 WGS 84/UTM zone 33 N Qgis version: QGIS 3.34 Prizren
